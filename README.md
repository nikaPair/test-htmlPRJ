<h2>Запуск проекта:</h2>

<h3>Клонирование репозитория:</h3>

Клонировать репозиторий себе на локальную машину с помощью команды:

git clone <URL репозитория>

Устанавливаем зависимости: 

npm install

Запуск проекта:

npm run start
